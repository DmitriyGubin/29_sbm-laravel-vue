Это проект с технологиям laravel + vue.js.
Привлекаются сторонние библиотеки - phpmailer, jquery, fancybox, axios и др.
Используется препроцессор SASS и сборщик модулей js - Vite.
Идёт взаимодействие со сторонними API - Wazzup, google/apiclient (интеграция с гугл таблицами), Bitrix24 REST API
Инфомация о поставщиках и технике подгружается из гугл таблицы раз в день с помощью Cron. 

Дамп базы данных - base.sql

Файлы инструкция для пользователей, инструкция для разработчиков и техзадание лежат в папке Docs.

Ссылка ---- https://examplesites.tw1.ru/ 
Это версия для тестов, на продакшине например анкета менеджера при регистрации попадает на модерацию к администратору. Пока анкета не будет подтверждена, менеджер не сможет зайти в свой кабинет.
